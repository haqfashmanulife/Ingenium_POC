About Owner Name/Insured Name Search:

Forward match search is performed for name fields. Please enter text exactly from the beginning.

Example: If you want to search "Manulife Japan Corp",

Good: "Manulife J"

Bad: "MLJ".



Note for Owner Name Search:

In case of Individual Owner, enter either Last Name or First Name, or both.

In case of Corporate Owner, enter Company Name.

Note for MGA Channel policies:

- Enter MGA Agent Office Code for "Servicing Agent ID"

- Search with MGA Agent employee Code/Name is not available.
